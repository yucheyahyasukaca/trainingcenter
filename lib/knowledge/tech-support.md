## Masalah Umum

### Login dan Akun
**Tanya:** Saya tidak bisa login.
**Jawab:**
1. Pastikan email dan password yang Anda masukkan sudah benar.
2. Jika Anda mendaftar menggunakan Google, silakan klik tombol "Masuk dengan Google".
3. Coba refresh halaman atau bersihkan cache browser Anda.
4. Jika masih gagal, gunakan fitur "Lupa Password" atau hubungi admin.

**Tanya:** Bagaimana cara reset password?
**Jawab:**
Saya bisa membantu Anda mereset password sekarang juga. Tolong sebukan **alamat email** yang Anda gunakan untuk akun Garuda Academy, dan saya akan mengirimkan link reset password ke email tersebut.

**Tanya:** Email di profil saya kosong setelah login Google.
**Jawab:**
Ini adalah masalah yang sudah kami perbaiki. Sistem akan otomatis menyinkronkan profil Anda saat Anda login kembali. Silakan logout dan login ulang.

### Pelatihan dan Kelas
**Tanya:** Bagaimana cara mendaftar pelatihan?
**Jawab:**
1. Masuk ke halaman Dashboard atau Program Pelatihan.
2. Pilih program yang ingin Anda ikuti.
3. Klik tombol "Daftar Sekarang".
4. Jika Anda memiliki kode referral, masukkan saat pendaftaran untuk mendapatkan poin.

**Tanya:** Saya sudah menyelesaikan kelas, tapi sertifikat belum muncul.
**Jawab:**
Sertifikat akan otomatis digenerate setelah Anda menyelesaikan semua materi dan lulus kuis (jika ada).
1. Cek tab "Sertifikat" di dashboard Anda.
2. Pastikan status enrollment Anda sudah "Completed".
3. Jika belum muncul, tunggu 5-10 menit dan refresh halaman.

### Masalah Teknis Website
**Tanya:** Saya mendapat error 502 Bad Gateway.
**Jawab:**
Error ini biasanya bersifat sementara karena masalah jaringan atau cache.
1. Coba refresh halaman (tekan F5 atau Ctrl+R).
2. Bersihkan cache browser Anda.
3. Coba akses menggunakan tab Incognito/Private.

**Tanya:** Halaman loading terus menerus.
**Jawab:**
Koneksi internet Anda mungkin tidak stabil. Periksa koneksi internet Anda. Jika internet lancar namun tetap loading, server kami mungkin sedang sibuk. Silakan coba lagi dalam beberapa saat.
