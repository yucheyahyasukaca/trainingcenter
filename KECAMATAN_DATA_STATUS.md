## Catatan Penting ⚠️
**Sistem Fallback Aktif**: File `app/register-referral/[code]/page.tsx` memiliki data `districtsData` yang berfungsi sebagai fallback. Jika data tidak ditemukan di `lib/data/kecamatan-indonesia.ts`, sistem akan otomatis menggunakan data dari `districtsData`.

Artinya, meskipun beberapa provinsi belum ditambahkan ke file utama, aplikasi tetap bisa berfungsi dengan data fallback yang sudah ada.

File `lib/data/kecamatan-indonesia.ts` sudah siap untuk menampung semua data. Data ditambahkan secara bertahap untuk menjaga performa aplikasi. Untuk penggunaan produksi, pertimbangkan untuk memindahkan data ke database atau API backend.
